## Local test
Double-click index.html and open in browser.

## GitHub Pages
1. Create a new GitHub repo called tenangkita-demo.
2. Upload index.html and the assets folder.
3. Go to Settings > Pages.
4. Source: Deploy from branch.
5. Branch: main.
6. Folder: /root.
7. Save.
8. Wait 1-3 minutes for the GitHub Pages URL.
Deployment trigger 2.
