# TenangKITA Portal v4.1 MYDS Alignment

Static GitHub Pages prototype for TenangKITA.

## Purpose

This version rebuilds TenangKITA as a portal first service instead of a mode based app screen.

## Key changes

1. Removed Mod Rakyat and Mod Kerajaan as the main interface pattern.
2. Replaced the old side app layout with a public portal navigation structure.
3. Added clear sections for Utama, Semak Keluarga, Harga & Peluang, Minyak & Perjalanan, Manfaat & Semakan Rasmi, Amanah Data and Untuk Agensi.
4. Kept citizen journey as the main journey while making agency and government value a dedicated section.
5. Added MYDS aligned masthead, phase banner, navbar, cards, tags, summary list, table, callout, footer, skip link and accessible form labels.
6. Added working static interactions for family summary, location permission state and fuel estimate.
7. Preserved safe wording such as pilihan harga berhampiran, mungkin berkaitan and semak rasmi.
8. Preserved data trust labels such as Data Rasmi, Data Langsung, Data Contoh, Anggaran, Semak Rasmi and Perlu Integrasi.
